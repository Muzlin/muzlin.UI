## 开始使用

1. 开启 border-box
    ```
    建(bi)议(xu)使用 border-box -- IE8以上的浏览器都支持
    ```
2. 安装 muzlin.ui
    ```
    npm i muzlin.ui
    ```
3. 引入 muzlin.ui
    ```
    import { Button, ButtonGroup, Icon } from 'muzlin.UI'
    import 'muzlin.ui/dist/index.css'

    export default {
      name: 'app',
      components: {
        'm-button': Button,
        'm-icon': Icon
      }
    }
    ```
